                    1.3.7.1  二元正态分布概率密度函数
函数名 
二元正态分布概率密度函数
变量 
x,y
表达式 
var xMean = 1;
var xStd = 2; 
var yMean = 2;
var yStd = 1; 
var coefficient = 0.2; 
var dx = x - xMean;
var dy = y - yMean;
var xyStd = xStd * yStd;
var dco = 1 - coefficient * coefficient;
var px = dx * dx / ( xStd * xStd );
var py = dy * dy / ( yStd * yStd );
var pxy = 2 * coefficient * dx * dy / xyStd;
var expP = - (px + py - pxy) / ( 2 * dco)
var div = 2 * Math.PI * xyStd * Math.sqrt(dco);
Math.exp(expP) / div
