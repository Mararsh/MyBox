图像的范围

1. 范围的作用

在处理图像时，可以利用图像的范围来选择需要处理的像素，即设置局部处理的条件。

2. 定义范围

范围由“形状的约束”和“匹配颜色列表”共同定义。

2.1 形状的约束

  1. “形状”限制像素的坐标。
  2. 选项“形状反选”表示形状以外的像素。

2.1.1 全图

图像的所有像素。

2.1.2 矩形、圆形、椭圆、多边形

  1. 拖动锚点以调整形状。
  2. 拖动形状以平移它。
  3. 右键点击图像以弹出形状的菜单。
  4. 对于多边形，左键点击图像以给多边形添加顶点。

2.1.3 抠图

  1. 点击图像，以添加点到起始点的列表。
  2. 区分八邻域抠图和四邻域抠图。

2.1.4 轮廓

  1. 选择一个带有透明背景的图像，它的轮廓被提取出来作为“形状”。
  2. 可以调整轮廓的长宽。

2.2 匹配颜色列表

  1. 用“形状的约束”选出来的像素，来匹配颜色列表。
  2. 选项“颜色反选”表示不匹配列表中任何颜色。

2.2.1 定义颜色列表

  1. 若颜色列表为空，则像素的颜色匹配总是成功。
  2. 添加颜色：点击颜色按钮以弹出调色板，或者选择“取色”然后点击图像。

2.2.2 匹配颜色的算法

  1. 当两个颜色之间的差别（即“颜色的距离”）小于给定的阈值时，则称为“颜色匹配”。
  2. 计算颜色的距离有多种算法可选，有的精确，有的性能好。
  3. 有的算法可以设置分量的权重，需要注意权重是正向的还是逆向的。
  4. 当用颜色列表来比较图像像素时，它们的关系是“或”，即像素与列表中任一颜色匹配时即为匹配成功。

2.3 其它选项

  1. 背景图像，可以是任意图像文件、也可为空。
    它不影响范围的使用，即按照一个图像定制的范围，可以应用在任意其它图像上。
  2. 选项“反选范围” ，表示选择范围之外的像素。
  3. 选项“处理透明像素”。

3. 显示范围

“蒙板图”用于突出显示选择的像素：

  1. 被选择的像素显示为原色。
  2. 不被选择的像素显示为混合了原色、蒙板的颜色、和蒙板透明度的颜色。
  3. 对于深色的图像，建议用浅色作为蒙板的颜色。
  4. 对于浅色的图像，建议用深色作为蒙板的颜色。
  5. 点击弹出按钮（CTRL+P）以显示原图、被选择的像素、和蒙板图。

4. 管理范围

  1. 在处理图像时，可以把选择像素时设置的条件保存为范围。
  2. 在处理图像时，可以把已保存的范围作为选择像素的条件。
  3. 范围以树形来保存和管理。
