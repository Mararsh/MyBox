图像的范围

1. 作用

在处理图像时，可以利用图像的范围来选择需要处理的像素，即设置局部处理的条件。

2. 定义范围

2.1 区域

  1. “区域”限制像素的坐标。
  2. 选项“区域反选”表示区域以外的像素。

2.2 颜色

  1. “颜色”匹配“区域”所选的像素。
  2. 添加颜色：点击颜色按钮以弹出调色板，或者选择“取色”然后点击图像。
  3. 选项“颜色反选”表示不匹配列表中任何颜色的像素。
  4. 如何匹配颜色的选项。

2.3 选择像素的其它选项

  1. 背景图像，可以是任意图像文件也可为空。
    它不影响范围的使用，即按照一个图像定制的范围，可以应用在任意其它图像上。
  2. 选项“反选范围” ，表示选择范围之外的像素。
  3. 选项“处理透明像素”。

3. 范围的类型

3.1 矩形、圆形、椭圆、多边形

  1. 拖动锚点以调整形状。
  2. 拖动形状以平移它。
  3. 右键点击图像以弹出形状的菜单。
  4. 设置颜色列表及其匹配选项。空的颜色列表表示不限制颜色。
  5. 对于多边形，左键点击图像以给多边形添加顶点。

3.2 抠图

  1. “区域”是用于抠图的起始点的列表。
  2. 点击图像以添加点。
  3. 算法只匹配图像本身的颜色，因此无需颜色列表。

3.3 颜色

  1. “区域”是所有像素。
  2. 按颜色列表匹配像素。

3.4 轮廓

  1. 选择一个带有透明背景的图像，它的轮廓被提取出来作为“区域”。
  2. 不匹配颜色，因此无需颜色列表。

4. 显示范围

“蒙板图”用于突出显示选择的像素：

  1. 被选择的像素显示为原色。
  2. 不被选择的像素显示为混合了原色、蒙板的颜色、和蒙板透明度的颜色。
  3. 对于深色的图像，建议用浅色作为蒙板的颜色。
  4. 对于浅色的图像，建议用深色作为蒙板的颜色。
  5. 点击弹出按钮（CTRL+P）以显示原图、被选择的像素、和蒙板图。

5. 管理范围

  1. 在处理图像时，可以把选择像素时设置的条件保存为范围。
  2. 在处理图像时，可以把已保存的范围作为选择像素的条件。
  3. 范围以树形来保存和管理。
